Searching Lace

This search page returns results from all pages in all OCR runs. Therefore, if you have two runs from the of the same set of images, you will get duplicates. Results from each run are grouped together.

Lace search is accent-sensitive but case-insensitive. So καὶ and καί are different, but καὶ and Καὶ are not.

Lace's search is based on the Lucene search engine. It supports:

  • Single character wildcard searches (a good work-around for accents): ξ?λο?ς
  • Multiple character wildcard searches: παρακυπτ* and αὐτ*ς (but not *όλεως
  • Fuzzy searches: εἰδώλων~ returns εἰδώλου, e.g.
  • Proximity searches: "εἰδώλων ἱερὰ"~10 for these within 10 words of each other
